Установка проекта
1)выполнить в терминале команду git clone https://github.com/luck456654/skillbox373.git находясь в пустой папке(в папке будующего проекта)
2)находясь в корневой папке проекта выполнить команду в терминале npm i
3)находясь в корневой папке проекта выполнить команду в терминале npm start
